# Portfolio Listing Copy

## Short Description
Freelance portfolio website with a production-style contact workflow: responsive frontend, validated Node.js API, SQLite lead storage, and an admin dashboard.

## Problem Solved
Small business sites often lose leads when forms are not persisted or tracked. This project captures inquiries reliably and provides a lightweight admin interface to review submissions.

## Key Outcomes
- Built end-to-end flow: UI -> API -> DB -> Admin
- Added server-side validation for cleaner lead data
- Added optional admin-token protection for leads endpoint

## Stack
HTML, CSS, JavaScript, Node.js, Express, SQLite

## Demo Talking Points
1. Submit a contact inquiry from the homepage.
2. Show persisted lead in `/admin.html`.
3. Explain validation and security boundary (`ADMIN_TOKEN`).
